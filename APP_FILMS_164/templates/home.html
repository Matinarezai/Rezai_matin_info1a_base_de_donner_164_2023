General Description of the Project

The database project involves developing an IT inventory software. The main objective is to allow storage of personal/employee hardware (computers, mice, keyboards, etc.), management of user licenses, office equipment (monitors, printers, etc.), and network hardware. The system must also allow for the creation and filling of different locations. It is important to be able to enter hardware into the database, assign it to a user and a location. Additionally, it should be possible to link components to hardware, for example, perform upgrades (RAM) on a laptop or change an SSD. Thus, hardware needs to be linked to a user and other hardware.

Database Fields: The database must include the following fields: Hardware, Users, Rest (structure).

Site Users with the Database: After the project is finalized, the IT inventory software will be used by the following entities within the company: Helpdesk Group, users outside the company, and affiliated companies.

The IT inventory software can also be used by companies affiliated with our company. It will provide valuable assistance by providing information on equipment present in workplaces or used by employees working from home, whose equipment belongs to the company. This will facilitate presentation to the insurance company to insure all these equipment.

The database will be very useful to know the users of software licenses outside the company and to identify the needs for licenses within the company. It will also help manage licenses based on ongoing projects, avoiding paying additional fees for unused licenses.
